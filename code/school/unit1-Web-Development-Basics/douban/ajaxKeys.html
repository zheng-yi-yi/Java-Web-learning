使用要点
  • Ajax的英文缩写是Asynchronous JavaScript and XML。使用JavaScript的异步请求和响应机制， 在不刷新整个页面的情况下局部更新页面的内容，实现动态加载数据、提高用户体验。
  • jQuery Ajax提供了方法$.ajax()，简化了Ajax的使用。
  • JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，它使用键值对的方式来表示数据，独立于编程语言， 易于阅读和编写，并且能够被大多数编程语言（后台编程语言和JavaScript脚本语言）轻松地处理。
  • JSON数据格式使用大括号{}表示一个对象，使用方括号[]表示一个数组。其中，键名必须使用双引号""括起来， 值可以是字符串、数字、布尔值、数组、对象或 null
  • 在浏览器里输入：https://movie.douban.com/ 访问豆瓣电影网站，选择某个电影后，可得到id（看浏览器地址栏subject）。
  • 访问：https://movie.querydata.org/api?id=10606004，可以得到电影相关信息的json格式数据。
  • 使用Postwoman，可测试豆瓣的后台接口并格式化返回的数据信息。
  • 访问网站 https://www.json.cn，可以在线解析json数据，以折叠式方式查看数据信息。
