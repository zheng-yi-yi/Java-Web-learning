客户端脚本使用要点
  • 客户端脚本用于实现交互效果或动态效果， 有原生JavaScript和jQuery库两种使用方式。
  • JavaScript常用的基本数据类型共有五种，分别是number（数值）、string（字符串）、boolean（逻辑真与假）、 undefined（未定义）和null（空值）。
  • JavaScript是一种弱类型语言，这意味着变量的数据类型可以在运行时动态地改变。
  • let和const是ES6引入的新特性，它们解决了使用var时存在的一些问题。
  • JavaScript可以使用对象类型（object），其赋值操作是按引用传递的。 数组与函数也属于对象类型，分别具有属性和方法。
  • 在 JavaScript 中，对象是基于原型（prototype）的，而不是基于类。 JavaScript内置的原型对象有Date、Array、String和Math等。 其中，前三需要使用new创建其实例。
  • 客户端脚本编程，通常还会使用浏览器对象，如window、document和location等。
  • 写客户端脚本程序，难免出现代码错误，可以使用控制台输出语句console.log(var)。打开浏览器调用程序， 查看浏览器控制台Console里的错误信息。
  • 前后端分离项目开发，可以使用jQuery Ajax。
  • jQuery库封装DOM元素为jQuery对象，简化了对DOM元素的使用。
