Web前端开发技术要点
  • 客户端浏览器程序可以解释HTML标签、CSS样式和客户端脚本JavaScript，并呈现页面效果。
  • 本页面综合使用了高效的弹性布局、常用的CSS+Div布局。
  • HTML标签用于产生页面元素；CSS样式用于修饰页面元素的外观或布局。
  • CSS基本选择器有：通用选择器、标签选择器、类选择器和id选择器。
  • 除了基本选择器外，还有组合选择器。最常用的选择器是后代选择器，它在多个基本选择器之间使用空格定义上下文关系。
  • HTML度量单位有以px表示绝对单位，也有以%或em表示的相对单位。其中，%是相对父容器的百分比；em是相对当前字体的位数。
  • 客户端脚本用于实现页面的交互效果或动态效果， 有原生JavaScript和jQuery框架两种使用方式， 将在本项目模块JavaScriptAndjQuery里介绍。
  • HTML元素外观不够美观，如按钮没有圆角。Element UI框架提供的元素组件的外观很漂亮。
  • 前后端分离项目开发，可以使用jQuery Ajax。
  • 基于Node环境的Web前端框架Vue，能很好地实现组件重用和路由等功能。
